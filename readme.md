# задание 1
## задача 2
### текст задачи
Составить программу, определяющую, в каком из данных двух чисел больше цифр.  
### Алгоритм решения
1) Пользователь вызывает: more_num(A, B)
2) Подсчёт количества цифр в числе A
(если число 0 то цифр 0, если не равно нулю то берется модуль, делится на 10 и
рекурсивно считается кол-во цифр)
3) Подсчёт количества цифр в числе B
4) Сравнение количества цифр и вывод результата

# задание 2
## задача 2
### текст задачи
Найти сумму тех элементов списка, которые оканчиваются на заданную цифру. 
### Алгоритм решения
1) Пользователь вызывает: sum_num(Список, D, S)
2) Если список пуст то сумма равно 0
3) разбиваем список на голову и хвост
4) для головы вычисляем последнюю цифру и сравниваем с заданной пользователем,
если совпадает - число добавляется к сумме, если нет - пропускается
5) повторяем для каждого элемента списка
6) выводим сумму

# задание 3
## задача 2
### текст задачи
Определим множество как список без повторяющихся элементов. Найти 
объединение множеств.
### Алгоритм решения
1) Если элемент совпадает с первым элементом списка то считается что элемент найден
Если не совпадает то проверка продолжается для оставшейся части списка
2) Если первый список пуст то результатом становится второй список
3) сли первый список не пуст то берётся первый элемент и отдельно рассматривается
оставшаяся часть списка
4) Если элемент уже есть во втором списке то он не добавляется в результат, а
программа продолжает работу с оставшейся частью первого списка
5) Если элемента нет во втором списке то он добавляется в результат, а 
программа продолжает работу с оставшейся частью первого списка
6) После обработки всех элементов формируется итоговый список без повторений
